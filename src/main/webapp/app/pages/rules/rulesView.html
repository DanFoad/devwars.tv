DevWars Rules

DevWars is still relatively new. As the game progresses, our rules will mature and become a lot more solid than they are now. Notice any inconsistencies? Let us know!

Classic Rules
  • HTML, CSS, JavaScript are allowed.
  • You are only allowed to code in your own workspace.
  • Libraries are allowed, but copying snippets from them are not.
  • No Frameworks of any kind are to be used.
  • The DevWars Twitch channel must not be open whilst competing.
  • No copying and pasting of code, howver copying code and URLs from teammates is acceptable.
  • No editing languages other than the one that you've been assigned.
  • After the 60 minute timer has run out, your code can no longer be modified.
  • Inline styling is not allowed. All styling must be performed using CSS.
Zen Garden Rules
  • Only CSS is allowed.
  • You are not allowed to modify the HTML template that is given to you.
  • No Frameworks of any kind are to be used.
  • The DevWars Twitch channel must not be open whilst competing.
  • No copying and pasting of code.
  • After the 60 minute timer has run out, your code can no longer be modified.
